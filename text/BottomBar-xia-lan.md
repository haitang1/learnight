# BottomBar组件  
## 概括：

**ButtomBar**定义一个高度42px,宽度100%的下栏。

```html
<bottom-bar>
    
</bottom-bar>
```
### 组件默认属性：
```css
.style{
    height:42px;
    overflow: hidden;
    position: absolute;
    width: 100%;
    bottom: 0;
    left:0;
    padding: 4px;
}
```
## 总结：

**ButtomBar**属性为固定值，定义整个页面的下栏，位置始终位于浏览器底部
