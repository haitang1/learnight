ToolBar按钮组

ToolBar组件

概括：

定义一组按钮，通过 left, right属性从从左右两边定义一组按钮

<toolbar
    :left="['按钮1'，'按钮2']"
    :right="['按钮3','按钮4']"
>
</toolbar>

总结：

了解更多请查看 Element

参数 说明 类型 可选值 默认值
left 向左边定义按钮 Array
right 向右边定义按钮 Array
