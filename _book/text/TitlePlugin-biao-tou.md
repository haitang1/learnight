# TitlePlugin组件
## 概括

定义表头文字的样式  


```html
    <title-plugin
    >
        表头文字
    </title-plugin>
```

### 默认样式
```css
.style title{
    border-bottom: 1px solid #409EFF;
    font-size: 14px;
    line-height: 22px;
    text-indent: 10px;
    color: #409EFF;
    font-weight: bold;
    margin: 10px auto;
}
```
